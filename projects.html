Language Agent Tree Search

I worked with Adam Hanka on designing and implementing AI agents and workflows that aid in decision support for financial questions. I combined both Language Agent Tree Search(LATS) and Retrieval Augmented Generation(RAG) to make an agent that can leverage both internet searches and private company data to make decisions. At the same time, providing a visualization of the agents thought process with a scoring system to decide the best response.


Hexagon: Tacit Knowledge Capture

Created RAG-based system to ask questions to add to knowledge base and answer questions based on retrieved data, then created structure on scraping tacit knowledge from metrologists in a non-intrusive way in order to build knowledge base. Worked with Hexagon staff and presented PoC to key stakeholders in Hexagon AB.
